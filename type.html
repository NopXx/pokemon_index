Setting Image Style

The Psychic type has few outright strengths, however, it also has few weaknesses. In the first generation it ended up being massively overpowered, mainly due to a complete lack of powerful Bug moves, its only weakness. Furthermore, a mistake in the game meant that Ghost-type moves had no effect on Psychic (although this only affected the low-powered Lick). Generation 2 rectified the situation with the addition of the Dark type along with better Pokémon and moves of all types.

Effects of the Psychic type

  • The power of Psychic type moves increases by 30% when Psychic Terrain is active.
  • The move Miracle Eye allows Psychic type moves to hit Dark type Pokémon.
  • The move Magic Powder changes the target's type to Psychic.
  • The items Mind Plate, Twisted Spoon and Odd Incense increase the power of Psychic type moves by 20% when held.
  • The Psychic Gem increases the power of a Psychic type move by 30% when held, and is then consumed.
  • The Payapa Berry, when held, neutralizes a super-effective Psychic type move.
  • Soul Dew boosts Psychic and Dragon type attacks by 20% when held by Latias or Latios.
